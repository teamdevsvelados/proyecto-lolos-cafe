# Lolo’s Café – Página Web

Proyecto académico en desarrollo para la creación de una plataforma web de una cafetería, desarrollado por el equipo **Devsvelados** de Generation MX.
Actualmente el proyecto cubre el frontend, con una evolución planeada hacia un backend en Java con Spring Boot y SQL.

## Descripción General

Lolo’s Café es una aplicación web orientada a clientes reales de una cafetería, cuyo objetivo es permitir la visualización del negocio, navegación del menú y, en futuras etapas, la gestión de pedidos.

El proyecto sirve como base de aprendizaje técnico y de trabajo colaborativo bajo metodología Scrum.

# Lolo’s Café – Website

Academic project in development for the creation of a web platform for a coffee shop, developed by the **Devsvelados** team from Generation MX. The project currently covers the frontend, with a planned evolution toward a backend using Java with Spring Boot and SQL.

## General Description

Lolo’s Café is a web application aimed at real coffee shop customers. Its goal is to allow business presentation, menu navigation, and, in future stages, order management.

The project serves as a foundation for technical learning and collaborative work under the Scrum methodology.

## Project Status
